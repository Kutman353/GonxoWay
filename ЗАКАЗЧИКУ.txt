Gonzo Way — как открыть сайт и принимать заявки
================================================

1) ПРОСМОТР У СЕБЯ НА КОМПЬЮТЕРЕ
   • Вариант А: в VS Code / Cursor откройте папку gonzo-way → правый клик по index.html → Open with Live Server.
     Тогда в браузере сразу откроется сайт, а не список папок.
   • Вариант Б: если Live Server запущен на родительской папке (newProject), откройте вручную:
     http://127.0.0.1:5500/gonzo-way/index.html
     Либо в корне newProject лежит index.html — он перенаправляет на gonzo-way.

2) ЗАЯВКИ С ФОРМЫ
   • Реальные заявки на почту приходят только после публикации сайта на Netlify (или другом хостинге с поддержкой Netlify Forms).
   • На Live Server и при открытии файла с диска форма не шлёт письма — включён демо-переход на страницу «Спасибо», чтобы ничего не «ломалось».

3) ЧТО СДЕЛАТЬ НА NETLIFY
   • Загрузите содержимое папки gonzo-way (или укажите в настройках сайта Base directory = gonzo-way).
   • После деплоя: Site configuration → Forms — должна появиться форма gonzo-quote.
   • Notifications — укажите email, куда слать заявки.
   • В футере сайта замените почту, телефон и ссылки на соцсети на ваши.

4) ПАПКА assets
   • К сайту Gonzo Way она не относится. Если в проекте есть лишняя папка — её можно не трогать или убрать из репозитория, на лендинг это не влияет.

5) КОМПОНЕНТЫ И FIGMA
   • Стили шапки и кнопок: файл components.css (палитра в :root).
   • Готовые HTML-фрагменты для переноса в макет: папка partials/ (header.html, buttons.html).
   • Фото The Nomad на сайте: Unsplash, жёлтый внедорожник в горах — в Figma можно вставить своё «сочное» фото брутального джипа с палаткой на крыше.
